# 夜读论文

时间：每周六或周日晚上8点到10点(约2小时)

地点：线上 zoom 会议（可能会录制视频）

概述：

0. 回顾上期的内容（5-10分钟）
1. 检索某主题，挑选一些分析文章，然后评选2篇文章来阅读（20分钟）
2. 开始第一次阅读论文（5-10分钟）
3. 开始第二次阅读论文
4. 第二次阅读论文完成之后，确定是否第三次阅读论文
5. 每一次会议都进行讨论和总结输出；

>每期阅读时，我们会将大家分别参考的资料链接贴到每期的 issue 链接上（在会议进行时方便大家点击查阅，同时也有记录了，不用单独再补充记录了）。

主题/论文：《xxxxxx》

是否开放：目前处于尝试阶段，暂时不对外开放，还请见谅。
